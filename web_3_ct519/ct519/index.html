CT519

In this section, I will summarize the knowledge and experiences gained while working on the CT519 exam project.

Summary of Knowledge

The CT519 project involved designing a Docker and Docker Compose setup with Nginx as a reverse proxy, distributing requests in a round-robin fashion among multiple web servers. The main concepts learned include:

  • Setting up Docker containers and Docker Compose
  • Configuring Nginx as a reverse proxy
  • Load balancing using Nginx
  • Creating and linking web services with different content

Implementation Steps

The project implementation involved the following steps:

  1. Creating Docker Compose configuration for multiple web services
  2. Setting up Nginx configuration for reverse proxy and load balancing
  3. Developing HTML content for each web service
  4. Testing the setup to ensure correct load balancing and content delivery

This project provided valuable hands-on experience with Docker, Nginx, and web server management.
